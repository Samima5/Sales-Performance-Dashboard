# Sales-Performance-Dashboard

End-to-end sales performance analysis using SQL, Python, and Power BI with interactive dashboards and KPIs.

## Project Details
This project analyzes sales performance across regions and products using SQL, Python, and Power BI. The dataset was cleaned and transformed using Power Query and Pandas. Key KPIs such as Total Sales, Profit, Monthly Trend, and Top Products were created. An interactive Power BI dashboard was designed to visualize YOY growth and region-wise performance. The final outcome helps in understanding business trends and making data-driven decisions.
